# 创世 / 四因创世（Genesis · Cosmogony）— 设计

## 概述

一个新的「玩法」页 `/genesis`，纯前端**概念/艺术原型**（星球与文案为占位内容，暂不接真实数据）。

核心叙事:四位古希腊原初神，按**亚里士多德四因说**分工协作，给同一批星球**逐层叠加**，把"一堆设定"变成"活的文明"。四神各掌一**层**，合起来正好是一张完整的「世界观知识图谱」。点击任一星球，回放它被四神**一层层创造**出来的过程。

| 神 | 因 | 层 | 含义 | 因之色 |
|---|---|---|---|---|
| 卡俄斯 Chaos | 质料因 | 规则层 | 世界由什么构成（schema 之根：允许哪些实体/关系） | 混沌紫 `#b388ff` |
| 盖亚 Gaia | 形式因 | 实体层 | 世界如何成形（地理/生态/物种等具体节点） | 生机青绿 `#3ee0b0` |
| 厄洛斯 Eros | 动力因 | 关系层 | 世界为何运转（实体间连接边:冲突/繁衍/依存） | 炽玫红 `#ff6b9d` |
| 谟涅摩绪涅 Mnemosyne | 目的/意义因 | 事件层 | 世界留下什么（带时间戳的事件/历史，叠在前三层之上） | 记忆金 `#ffd166` |

> 注:厄洛斯指赫西俄德《神谱》中的原初厄洛斯（创世三巨头之一），非小爱神。

这套结构与 Atlas 自身同构（契约 schema → 规则、items → 实体、graph edges → 关系、timeline → 事件），为日后接真实数据留好接口；本期不实现。

## 目标 / 非目标

**目标**
- 做出一个视觉惊艳、可评审的前端 UI 原型,讲清"四智者创世"的叙事。
- 高潮交互:点击星球 → 创世动画逐层重演。
- 零依赖(原生 HTML/CSS/JS),贴合 Atlas 服务端渲染、无构建链的风格。
- 复用宇宙系列的深空视觉语言,并叠加古典金的"古希腊"气质。

**非目标(本期不做)**
- 不接真实数据、不碰数据库/契约/嵌入;星球与文案硬编码。
- 不引入 Three.js 或任何前端库/构建步骤。
- 四神星座为**抽象示意形象**,不追求写实人像。

## 页面身份

- 路由:`GET /genesis`,纯静态模板渲染(无数据查询)。
- 中文名 **创世**;眉标 `COSMOGONY · 四因创世`。
- 导航:加入 `_topbar.html` 的「玩法」下拉(放在图谱之后或列表首位,作为旗舰玩法)。
- 版式:全宽深空场景,`main` 撑满(用 `main:has(.genesis)` 解除 1100px 限制,参照 Hermes 页),场景高度 `calc(100dvh - topbar)`。

## 配色(深空 × 古典金)

```
ground:   #080b16   深空底（复用宇宙系列）
text:     #e6ebf5   冷白正文
gold:     #c9a86a   结构性古典金（回纹/边框/标签/星座底线/铭文）
gold-hi:  #f0dca0   高光金（铭文点亮、回纹流光）
```

四因之色见上表（`#b388ff` / `#3ee0b0` / `#ff6b9d` / `#ffd166`）。

- 暖角晕:在冷的深空底上,四角各叠一团**极淡的对应因之色**辉光(rest 态约 6–8% 不透明度),让四角"有神在"。
- 创世动画中,当前相位会让**整屏微微泛该神之色**(scene 叠一层 `mix-blend` 的色雾,峰值约 10–14%),相位切换时回落。

```
┌───────────────────────────────────────────────────────────┐
│ ◆卡俄斯·星座            COSMOGONY · 四因创世            盖亚·星座◆ │
│  质料因 / 规则层               创 世                  形式因 / 实体层 │
│                                                             │
│              ·   ◦    ●    ·    ◦         ← 中央漂浮星球场       │
│           ◦       ●       ·    ◦    ●                         │
│              ·    ◦    ●      ·                              │
│                                                             │
│ ◆厄洛斯·星座                                       谟涅摩绪涅·星座◆ │
│  动力因 / 关系层                                  目的/意义因 / 事件层 │
└───────────────────────────────────────────────────────────┘
        （四角内描金回纹 meander 包边，是全页唯一的古典签名）
```

- **标题**:顶部居中,眉标(等宽金,letter-spacing 宽) + 渐变大标题「创世」。属于 rest 态,进入创世动画时淡出/下沉,避免与高潮抢戏。
- **四角·神(星座线稿)**:每位神是一组星点(8–14 颗)+ 细线连成的**抽象星座形象**,线/点染该神之色,默认偏暗(约 45% 亮度)。
  - 草图意象(SVG 手工点集,抽象示意即可):
    - 卡俄斯 = 向内螺旋汇聚的散乱星群(从无序收束)。
    - 盖亚 = 环抱的母体/山-球轮廓(稳定的三角+弧)。
    - 厄洛斯 = 展翼/交缠双弧(动态的 V 与对称弧)。
    - 谟涅摩绪涅 = 单臂螺旋(记忆之旋)。
  - 每幅旁标:神名(希腊 + 中文)、因、层。
  - **hover**:整幅星座点亮(亮度→100% + 辉光),星点轻微涨缩,并朝中央射出一道**淡色光束**(暗示"在喂养世界");同时该角的暖晕增强。光束为装饰,不改变星球。
- **中央·星球场**:约 16 颗星球,缓慢漂浮(轻微平移 + 视差,分前后两层做景深)。每颗:径向渐变球体 + 柔辉光,大小/微色各异,带一个诗意**世界名**(占位,见下)。hover 时该球略放大 + 名字浮现。

## Rest 态氛围(克制但活)

- 复用宇宙系列**星尘**层(呼吸闪烁)。
- 星球**视差漂浮**:前景球漂得快、后景慢,营造深度。
- 偶发**流星**(每 8–14s 一道,斜划而过,纯 CSS 动画)。
- 四角→中央的**持续微光束**(极淡、缓脉动),暗示创世从未停止。
- `prefers-reduced-motion`:关闭漂浮/流星/脉动,星尘静止。

## 高潮:创世动画重演（点击星球）

点击一颗星球触发,由一个小 JS 状态机驱动相位 class:

0. **入场**:选中球**飞向中央并放大**,其余星球与标题淡出/退后(scene 暗下来,聚光中央)。底部出现**四颗因色进度点**。
1. **相位一 · 卡俄斯（紫,规则层）**:左上角卡俄斯星座**炸亮**→ 一束紫色光束 + 一群**混沌粒子**从该角汇入中央,凝成一颗**线框规则球**(经纬网格/wireframe)。铭文字幕浮现:`质料因 · 规则层 ——「世界由什么构成」`。
2. **相位二 · 盖亚（青绿,实体层）**:盖亚星座炸亮 → 线框球表面长出**地壳纹理**,球面浮现一批**实体节点**(发光小点 = 地理/物种)。字幕:`形式因 · 实体层 ——「世界如何成形」`。
3. **相位三 · 厄洛斯（玫红,关系层）**:厄洛斯星座炸亮 → 实体节点间连出**关系弧线**(edges),球体开始**自转**(世界动起来)。字幕:`动力因 · 关系层 ——「世界为何运转」`。
4. **相位四 · 谟涅摩绪涅（金,事件层）**:记忆星座炸亮 → 一圈带**时间刻度的历史环**缠上球体并缓慢游动,世界"活"了。字幕:`目的/意义因 · 事件层 ——「世界留下什么」`。
5. **收尾(成品态)**:四层共显的"活世界"停在中央,四神全亮,左下角小图例(四因四色四层),标题区出现该世界名。

- **节奏**:每相位约 1.0–1.3s,含相位间一个短促的**白/金闪光**过场(big-bang 感);全程约 4.5–5.5s。
- **相位泛色**:进行中的相位让整屏微泛该神之色(见配色)。
- **铭文字幕**:像碑刻——古典金、宽字距、淡入,自带细下划金线;切相位时上一条淡出。
- **控件**:右下 `重放` / `跳过`(跳过直接到成品态);关闭 `✕ / Esc / 点空白` → 回到星海(标题与星球场淡回)。
- `prefers-reduced-motion`:跳过逐帧编排,**直接呈现成品态**(四层静态共显),字幕一次性显示。

## "酷"的强化层（把大胆押在高潮，别处克制）

- **唯一古典冒险**:内描金**回纹 meander**(希腊钥匙纹)包在场景四角,作为"古希腊 × 宇宙"的签名;rest 态静默,创世过场时有一缕金色流光顺纹路跑一圈。
- **电影感**:入场聚光压暗四周(vignette 加深)、相位切换的闪光过场、成品态的缓缓"呼吸"。
- 这些都为高潮服务;rest 态与四角保持安静、精确的留白与对齐。

## 组件 / 文件结构

- `src/aishelf/site/templates/genesis.html` —— 页面标记 + 在内容块内 `<link rel="stylesheet" href="/static/genesis.css">` 与 `<script src="/static/genesis.js">`。
- `src/aishelf/site/static/genesis.css` —— 本页全部样式(深空场景、四角神、星球、创世动画 keyframes、回纹、reduced-motion)。
- `src/aishelf/site/static/genesis.js` —— 场景数据(硬编码星球/世界名)、漂浮/流星氛围、创世动画状态机、控件与键盘交互。
- `src/aishelf/site/templates/base.html` —— 增加 `{% block head %}{% endblock %}`(在 `</head>` 前),供本页注入样式;不影响其它页。
  - 备选:直接在内容块里放 `<link>`(浏览器容忍 body 内 link),则不动 base.html。**采用 head block 方案**(更规范)。
- `src/aishelf/site/app.py` —— 新增 `@app.get("/genesis", response_class=HTMLResponse)`,仅渲染模板(无数据)。
- `src/aishelf/site/templates/_topbar.html` —— 「玩法」下拉加入 `<a href="/genesis">创世</a>`(列首,旗舰位)。

## 测试

- `tests/unit/` 加一个冒烟测试:`GET /genesis` 返回 200 且页面含关键标记(如 `genesis`、`创世`、四神名其一)。静态页无需更多逻辑测试。
- 视觉:实现后用无头 Chrome 截图核验 rest 态 + 创世动画的关键相位(注入相位 class 截图)。

## 无障碍

- `prefers-reduced-motion: reduce`:关闭漂浮/流星/脉动/逐帧创世,直达成品态。
- 星球可聚焦(`tabindex`),`Enter` 触发创世,`Esc` 关闭;四角神 `aria-label` 标注神/因/层。
- 铭文与四角文字保证在深底上的对比度(古典金 `#c9a86a` 在 `#080b16` 上达标)。
