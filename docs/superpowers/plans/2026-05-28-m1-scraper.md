# M1 Scraper Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 实现 aishelf 项目的 M1 Scraper —— 一个 CLI 工具,从 YouTube + Bilibili 按账号 + 日期区间抓取 AI 领域视频的元数据、封面、字幕/转录,落盘成稳定的文件结构供 M2 消费。

**Architecture:** Strategy 模式 + 同步执行。`PlatformAdapter` Protocol 定义平台抽象,YouTube/Bilibili 各一份实现。顶层 pipeline 按账号迭代,逐视频"列表 → 取元 → 取封面 → 取字幕(无则下临时音频走 Whisper)→ 原子落盘"。单视频失败不阻塞批次,失败记录单独存到 `data/_failures/`。

**Tech Stack:** Python 3.11+,yt-dlp,youtube-transcript-api,openai SDK,httpx,PyYAML,pydantic v2;pytest + respx + monkeypatch 测试,Whisper API 与 yt-dlp 永远 mock。

**Spec:** [`docs/superpowers/specs/2026-05-28-m1-scraper-design.md`](../specs/2026-05-28-m1-scraper-design.md)

```markdown
# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project Overview

**aishelf** aggregates AI-domain video content (interviews, technical talks) from multiple platforms. It is composed of three modules with stable file/DB interfaces between them:

| Module | Role | Status |
|---|---|---|
| **M1 Scraper** | Crawl YouTube + Bilibili by author + date range; output transcripts/meta/thumbnails to filesystem | In development |
| **M2 Understanding** | Use LLMs to extract structured topics/summaries from transcripts, save to DB | Planned |
| **M3 Web** | Frontend + API for browsing aggregated content | Planned |

Each module has its own spec in `docs/superpowers/specs/` and plan in `docs/superpowers/plans/`. **Implementation order is M1 → M2 → M3.**

## Commands

- Install with dev deps: `pip install -e ".[dev]"`
- Run M1: `python -m aishelf.scraper run --since YYYY-MM-DD`
- Validate config: `python -m aishelf.scraper validate-config`
- Run tests (network skipped by default): `pytest`
- Run including network smoke tests: `pytest -m ""`
- Run a single test: `pytest tests/unit/test_models.py::test_specific -v`

## Architecture

Source layout uses a `src/` directory; package is `aishelf`. M1 lives in `src/aishelf/scraper/`.

**Key design points for M1** (see spec for the full picture):

- Strategy pattern with `PlatformAdapter` Protocol in `adapters/base.py`; one implementation per platform (`youtube.py`, `bilibili.py`)
- Synchronous execution — one video at a time. Per-video failure does not block the batch.
- Output contract: `data/<platform>/<account_id>/<video_id>/{meta.json, thumb.<ext>, transcript.json}`. Failures go to `data/_failures/<platform>/<account_id>/<video_id>.json`.
- Atomic writes: write into `<final_dir>.tmp`, then `os.rename`. Existence of `meta.json` is the success signal.
- Transcripts: prefer platform CC; if missing, fetch temporary audio and call OpenAI Whisper API. Audio is deleted after transcription.
- Whisper API and yt-dlp are **always mocked in tests**. Live calls only run under the `network` pytest marker.

## Config

`config/authors.yaml` is the user-edited account list (gitignored). `config/authors.example.yaml` is committed as a template. Each entry is a single platform account; the same person on two platforms is two entries.
```

```python
"""Pydantic data models used across the M1 Scraper module."""
from __future__ import annotations

from datetime import datetime
from typing import Literal

from pydantic import BaseModel, ConfigDict, Field, field_validator


def _require_tzaware(dt: datetime) -> datetime:
    if dt.tzinfo is None:
        raise ValueError("datetime must be timezone-aware (use UTC)")
    return dt


class VideoListing(BaseModel):
    """Lightweight per-video record returned by adapter.list_videos().

    Just enough to decide whether to fetch or skip.
    """

    model_config = ConfigDict(extra="forbid")

    video_id: str
    title: str
    published_at: datetime

    @field_validator("published_at")
    @classmethod
    def _tz(cls, v: datetime) -> datetime:
        return _require_tzaware(v)
```

```python
class AccountRef(BaseModel):
    """Reference to the platform account that uploaded the video."""

    model_config = ConfigDict(extra="forbid")

    id: str
    display_name: str


Platform = Literal["youtube", "bilibili"]


class VideoMeta(BaseModel):
    """Full metadata for one video — what gets written to meta.json.

    Schema version is part of the contract with M2. Bump when adding fields.
    """

    model_config = ConfigDict(extra="forbid")

    schema_version: int = 1
    platform: Platform
    video_id: str
    url: str
    account: AccountRef
    title: str
    description: str
    published_at: datetime
    duration_seconds: int
    thumbnail_path: str
    transcript_path: str
    view_count: int | None
    tags: list[str] = Field(default_factory=list)
    fetched_at: datetime

    @field_validator("published_at", "fetched_at")
    @classmethod
    def _tz(cls, v: datetime) -> datetime:
        return _require_tzaware(v)

    @field_validator("published_at", "fetched_at", mode="after")
    @classmethod
    def _serialize_to_z(cls, v: datetime) -> datetime:
        # Normalize to UTC so model_dump(mode="json") yields ...Z suffix.
        return v.astimezone(tz=v.tzinfo)
```

For ISO 8601 with `Z` suffix on `model_dump(mode="json")`, pydantic by default emits `+00:00`. Override with a serializer:

Add to the imports:

```python
from pydantic import field_serializer
```

And replace the `_serialize_to_z` validator block with:

```python
    @field_serializer("published_at", "fetched_at", when_used="json")
    def _to_z(self, v: datetime) -> str:
        return v.astimezone(tz=v.tzinfo).strftime("%Y-%m-%dT%H:%M:%SZ") if v.tzinfo.utcoffset(v).total_seconds() == 0 else v.isoformat()
```

Simpler version — assume always UTC and emit Z:

```python
    @field_serializer("published_at", "fetched_at", when_used="json")
    def _to_z(self, v: datetime) -> str:
        return v.strftime("%Y-%m-%dT%H:%M:%SZ")
```

Use this simpler form. The validator already forces tzaware; emitting `Z` assumes UTC. Add a stricter check that enforces UTC:

Update `_require_tzaware`:

```python
def _require_tzaware_utc(dt: datetime) -> datetime:
    if dt.tzinfo is None:
        raise ValueError("datetime must be timezone-aware (UTC)")
    if dt.utcoffset().total_seconds() != 0:
        raise ValueError("datetime must be in UTC (offset 0)")
    return dt
```

And rename the validator helper call to `_require_tzaware_utc`. Update both `VideoListing` and `VideoMeta` validators to use it.

```python
"""Tiny retry helper with exponential backoff.

Used by adapters and transcribe.py to wrap flaky external calls.
"""
from __future__ import annotations

import logging
import time
from typing import Callable, TypeVar

logger = logging.getLogger(__name__)

T = TypeVar("T")


class RetryError(Exception):
    """Raised after all retry attempts are exhausted."""


def retry(
    fn: Callable[[], T],
    *,
    attempts: int = 3,
    base: float = 1.0,
    retry_on: tuple[type[BaseException], ...] = (Exception,),
) -> T:
    """Call ``fn()``, retrying with exponential backoff on listed exceptions.

    Backoff between attempt N and N+1 is ``base * 2**N`` seconds.
    Exceptions not listed in ``retry_on`` propagate immediately.
    """
    last: BaseException | None = None
    for attempt in range(attempts):
        try:
            return fn()
        except retry_on as e:
            last = e
            if attempt == attempts - 1:
                break
            delay = base * (2**attempt)
            logger.debug(
                "retry attempt=%d/%d sleep=%.2fs error=%s",
                attempt + 1,
                attempts,
                delay,
                e,
            )
            time.sleep(delay)
    raise RetryError(f"exhausted {attempts} attempts") from last
```

```python
"""Filesystem store for scraper outputs.

Layout:
    data_root/
        <platform>/<account_id>/<video_id>/
            meta.json
            thumb.<ext>
            transcript.json
        _failures/
            <platform>/<account_id>/<video_id>.json

Writes are atomic: contents are staged in <video_id>.tmp/ and renamed to
<video_id>/ only after all required files are present.
"""
from __future__ import annotations

import json
import os
from dataclasses import dataclass
from pathlib import Path

from aishelf.scraper.models import (
    FailureRecord,
    Platform,
    TranscriptResult,
    VideoMeta,
)

META_FILENAME = "meta.json"
TRANSCRIPT_FILENAME = "transcript.json"


@dataclass
class Store:
    data_root: Path

    # --- path helpers -----------------------------------------------------

    def video_dir(self, platform: Platform, account_id: str, video_id: str) -> Path:
        return self.data_root / platform / account_id / video_id

    def failure_path(self, platform: Platform, account_id: str, video_id: str) -> Path:
        return self.data_root / "_failures" / platform / account_id / f"{video_id}.json"

    # --- existence checks -------------------------------------------------

    def is_complete(self, platform: Platform, account_id: str, video_id: str) -> bool:
        return (self.video_dir(platform, account_id, video_id) / META_FILENAME).is_file()

    def has_failure(self, platform: Platform, account_id: str, video_id: str) -> bool:
        return self.failure_path(platform, account_id, video_id).is_file()
```

```python
    # --- atomic write -----------------------------------------------------

    def write_video(
        self,
        *,
        meta: VideoMeta,
        transcript: TranscriptResult,
        thumb_bytes: bytes,
        thumb_extension: str,
    ) -> None:
        """Write meta + transcript + thumb atomically.

        Stages into ``<video_id>.tmp/`` and renames to ``<video_id>/``.
        If a failure record for this video exists, it is removed.
        """
        final = self.video_dir(meta.platform, meta.account.id, meta.video_id)
        staging = final.with_suffix(".tmp")
        if staging.exists():
            _rmtree(staging)
        staging.mkdir(parents=True, exist_ok=True)

        # Ensure meta.thumbnail_path matches the actual filename we write.
        expected_thumb_name = f"thumb{thumb_extension}"
        if meta.thumbnail_path != expected_thumb_name:
            meta = meta.model_copy(update={"thumbnail_path": expected_thumb_name})

        (staging / expected_thumb_name).write_bytes(thumb_bytes)
        (staging / TRANSCRIPT_FILENAME).write_text(
            transcript.model_dump_json(indent=2), encoding="utf-8"
        )
        (staging / META_FILENAME).write_text(
            meta.model_dump_json(indent=2), encoding="utf-8"
        )

        if final.exists():
            _rmtree(final)
        os.rename(staging, final)

        # Clean up corresponding failure marker, if any
        fail = self.failure_path(meta.platform, meta.account.id, meta.video_id)
        if fail.is_file():
            fail.unlink()
            _prune_empty_parents(fail.parent, stop=self.data_root / "_failures")

    # --- failures ---------------------------------------------------------

    def write_failure(self, fr: FailureRecord) -> None:
        path = self.failure_path(fr.platform, fr.account_id, fr.video_id)
        path.parent.mkdir(parents=True, exist_ok=True)
        tmp = path.with_suffix(".json.tmp")
        tmp.write_text(fr.model_dump_json(indent=2), encoding="utf-8")
        os.rename(tmp, path)

    def remove_failure(self, platform: Platform, account_id: str, video_id: str) -> None:
        path = self.failure_path(platform, account_id, video_id)
        if path.is_file():
            path.unlink()
            _prune_empty_parents(path.parent, stop=self.data_root / "_failures")

    def iter_failures(self) -> "Iterable[FailureRecord]":
        root = self.data_root / "_failures"
        if not root.is_dir():
            return
        for path in root.rglob("*.json"):
            yield FailureRecord.model_validate_json(path.read_text(encoding="utf-8"))


# --- module-level helpers ---------------------------------------------------

def _rmtree(p: Path) -> None:
    import shutil
    shutil.rmtree(p, ignore_errors=False)


def _prune_empty_parents(start: Path, stop: Path) -> None:
    """Walk up from ``start`` removing empty directories, stopping at ``stop``."""
    current = start
    while current != stop and current.is_dir():
        try:
            current.rmdir()
        except OSError:
            return
        current = current.parent
```

```python
"""OpenAI Whisper API wrapper.

Wraps the OpenAI SDK so callers get a TranscriptResult directly. Maps the
Whisper "language" field (e.g. "english", "chinese") to BCP-47 short codes.

Whisper API is always mocked in tests — never call the real endpoint in CI.
"""
from __future__ import annotations

import logging
from dataclasses import dataclass
from pathlib import Path
from typing import Any

from aishelf.scraper.models import TranscriptResult, TranscriptSegment

logger = logging.getLogger(__name__)

# Whisper returns full English language names; normalize to BCP-47.
_LANG_NAME_TO_CODE = {
    "english": "en",
    "chinese": "zh",
    "japanese": "ja",
    "korean": "ko",
    "spanish": "es",
    "french": "fr",
    "german": "de",
}


@dataclass
class WhisperApi:
    """OpenAI Whisper API client.

    Pass a pre-built OpenAI client (real or fake). If omitted, a real one is
    constructed from the OPENAI_API_KEY environment variable.
    """

    client: Any = None
    model: str = "whisper-1"

    def __post_init__(self) -> None:
        if self.client is None:
            from openai import OpenAI  # imported lazily to keep tests light
            self.client = OpenAI()

    def transcribe(
        self, audio_path: Path, *, language_hint: str | None
    ) -> TranscriptResult:
        with open(audio_path, "rb") as f:
            response = self.client.audio.transcriptions.create(
                model=self.model,
                file=f,
                response_format="verbose_json",
                language=language_hint,
            )

        lang_raw = (getattr(response, "language", None) or "").lower()
        lang = _LANG_NAME_TO_CODE.get(lang_raw, lang_raw or "und")

        segments = [
            TranscriptSegment(
                start=float(s["start"]), end=float(s["end"]), text=s["text"].strip()
            )
            for s in (getattr(response, "segments", None) or [])
        ]
        full_text = (getattr(response, "text", None) or "").strip()

        return TranscriptResult(
            schema_version=1,
            language=lang,
            source="whisper_api",
            segments=segments,
            full_text=full_text,
        )
```

Note: the SDK's verbose_json response actually returns `segments` as a list of dicts when using the recent SDK. The code above handles both attribute and dict access uniformly (via `getattr` on a `SimpleNamespace` in tests, and on the real SDK object in prod). If the real SDK returns segments as dicts (not objects), the `s["start"]` access in segments comprehension works for both: a `dict.__getitem__` and an `SimpleNamespace.__getitem__` would fail. The fake uses `SimpleNamespace(**dict)` where segments is a list of dicts, so `s["start"]` is dict access — OK. For the real SDK, segments items are dicts in the verbose_json mode — also OK.

```python
"""YouTube platform adapter.

Uses yt-dlp for channel/video metadata and audio download, and
youtube-transcript-api for fast subtitle fetching.

External libraries are wrapped behind private methods (_ytdlp_extract_info,
_fetch_transcript_via_api) so tests can monkeypatch a single seam.
"""
from __future__ import annotations

import logging
import re
from datetime import date, datetime, timezone
from pathlib import Path
from typing import Any, ClassVar, Iterator

import httpx

from aishelf.scraper.adapters.base import PlatformAdapter
from aishelf.scraper.models import (
    AccountRef,
    Platform,
    TranscriptResult,
    TranscriptSegment,
    VideoListing,
    VideoMeta,
)

logger = logging.getLogger(__name__)


_HANDLE_RE = re.compile(r"^@[A-Za-z0-9_.-]+$")
_CHANNEL_URL_RE = re.compile(
    r"https?://(?:www\.)?youtube\.com/(?:channel/(?P<id>UC[\w-]+)|@(?P<handle>[\w.-]+))"
)


class YouTubeAdapter:
    platform: ClassVar[Platform] = "youtube"

    def resolve_account_id(self, raw: str) -> str:
        # Stable channel ID: pass through
        if raw.startswith("UC") and len(raw) >= 24:
            return raw

        # URL form
        m = _CHANNEL_URL_RE.match(raw)
        if m and m.group("id"):
            return m.group("id")

        # @handle or channel URL with handle → probe via yt-dlp
        url: str
        if _HANDLE_RE.match(raw):
            url = f"https://www.youtube.com/{raw}"
        elif m and m.group("handle"):
            url = raw
        else:
            raise ValueError(f"could not parse YouTube account input: {raw!r}")

        info = self._ytdlp_extract_info(url, opts={"extract_flat": True, "skip_download": True})
        channel_id = info.get("channel_id")
        if not channel_id:
            raise ValueError(f"could not resolve channel ID from {raw!r}")
        return channel_id

    # --- thin seams over external libs (mockable in tests) ----------------

    def _ytdlp_extract_info(self, url: str, *, opts: dict[str, Any]) -> dict[str, Any]:
        """Thin wrapper over yt_dlp.YoutubeDL.extract_info for monkeypatching."""
        from yt_dlp import YoutubeDL

        merged = {"quiet": True, "no_warnings": True, **opts}
        with YoutubeDL(merged) as ydl:
            return ydl.extract_info(url, download=False) or {}
```

```python
    def list_videos(
        self, account_id: str, since: date, until: date, limit: int
    ) -> Iterator[VideoListing]:
        url = f"https://www.youtube.com/channel/{account_id}/videos"
        info = self._ytdlp_extract_info(
            url, opts={"extract_flat": True, "skip_download": True, "playlistend": limit * 4}
        )
        # info["entries"] is newest-first

        since_ts = datetime(since.year, since.month, since.day, tzinfo=timezone.utc).timestamp()
        until_ts = (
            datetime(until.year, until.month, until.day, 23, 59, 59, tzinfo=timezone.utc).timestamp()
        )

        yielded = 0
        for entry in info.get("entries") or []:
            ts = entry.get("timestamp")
            if ts is None:
                continue
            if not (since_ts <= ts <= until_ts):
                continue

            published_at = datetime.fromtimestamp(ts, tz=timezone.utc)
            yield VideoListing(
                video_id=entry["id"],
                title=entry.get("title") or "",
                published_at=published_at,
            )
            yielded += 1
            if yielded >= limit:
                return
```

```python
        elif "png" in ct:
            ext = ".png"
```

```python
"""Bilibili platform adapter.

Anonymous fetching — no login or cookies. Most public AI-domain content is
accessible without auth. yt-dlp handles channel listing, video metadata,
and audio. Subtitles come from yt-dlp's "subtitles" dict (B站 only has
manually uploaded subs, no auto-generated).
"""
from __future__ import annotations

import logging
import re
from datetime import date, datetime, timezone
from pathlib import Path
from typing import Any, ClassVar, Iterator

import httpx

from aishelf.scraper.adapters.base import PlatformAdapter
from aishelf.scraper.models import (
    AccountRef,
    Platform,
    TranscriptResult,
    TranscriptSegment,
    VideoListing,
    VideoMeta,
)

logger = logging.getLogger(__name__)


_NUMERIC_RE = re.compile(r"^\d+$")
_SPACE_URL_RE = re.compile(r"https?://space\.bilibili\.com/(?P<id>\d+)")


class BilibiliAdapter:
    platform: ClassVar[Platform] = "bilibili"

    def __init__(self) -> None:
        self._info_cache: dict[str, dict[str, Any]] = {}

    def resolve_account_id(self, raw: str) -> str:
        if _NUMERIC_RE.match(raw):
            return raw
        m = _SPACE_URL_RE.match(raw)
        if m:
            return m.group("id")
        raise ValueError(f"could not parse Bilibili account input: {raw!r}")

    def list_videos(
        self, account_id: str, since: date, until: date, limit: int
    ) -> Iterator[VideoListing]:
        url = f"https://space.bilibili.com/{account_id}/video"
        info = self._ytdlp_extract_info(
            url, opts={"extract_flat": True, "skip_download": True, "playlistend": limit * 4}
        )
        since_ts = datetime(since.year, since.month, since.day, tzinfo=timezone.utc).timestamp()
        until_ts = (
            datetime(until.year, until.month, until.day, 23, 59, 59, tzinfo=timezone.utc).timestamp()
        )

        yielded = 0
        for entry in info.get("entries") or []:
            ts = entry.get("timestamp")
            if ts is None or not (since_ts <= ts <= until_ts):
                continue
            yield VideoListing(
                video_id=entry["id"],
                title=entry.get("title") or "",
                published_at=datetime.fromtimestamp(ts, tz=timezone.utc),
            )
            yielded += 1
            if yielded >= limit:
                return

    def fetch_meta(
        self, video_id: str, account_id: str, account_display_name: str
    ) -> VideoMeta:
        url = f"https://www.bilibili.com/video/{video_id}"
        info = self._ytdlp_extract_info(url, opts={"skip_download": True})
        self._info_cache[video_id] = info

        ts = info.get("timestamp")
        published_at = datetime.fromtimestamp(ts, tz=timezone.utc) if ts else datetime.now(tz=timezone.utc)

        return VideoMeta(
            schema_version=1,
            platform="bilibili",
            video_id=video_id,
            url=info.get("webpage_url") or url,
            account=AccountRef(id=account_id, display_name=account_display_name),
            title=info.get("title") or "",
            description=info.get("description") or "",
            published_at=published_at,
            duration_seconds=int(info.get("duration") or 0),
            thumbnail_path="thumb.jpg",
            transcript_path="transcript.json",
            view_count=info.get("view_count"),
            tags=list(info.get("tags") or []),
            fetched_at=datetime.now(tz=timezone.utc),
        )

    def fetch_thumbnail(self, meta: VideoMeta) -> tuple[bytes, str]:
        info = self._info_cache.get(meta.video_id) or self._ytdlp_extract_info(
            f"https://www.bilibili.com/video/{meta.video_id}",
            opts={"skip_download": True},
        )
        thumb_url = info.get("thumbnail")
        if not thumb_url:
            raise RuntimeError(f"no thumbnail URL for {meta.video_id}")
        resp = httpx.get(thumb_url, timeout=30.0, follow_redirects=True)
        resp.raise_for_status()
        ct = (resp.headers.get("content-type") or "").lower()
        ext = ".webp" if "webp" in ct else ".png" if "png" in ct else ".jpg"
        return resp.content, ext

    def fetch_subtitle(self, meta: VideoMeta) -> TranscriptResult | None:
        info = self._info_cache.get(meta.video_id)
        if info is None:
            info = self._ytdlp_extract_info(
                f"https://www.bilibili.com/video/{meta.video_id}", opts={"skip_download": True}
            )
            self._info_cache[meta.video_id] = info

        subs = info.get("subtitles") or {}
        if not subs:
            return None

        # Pick the first language available. Prefer zh-CN > zh > anything.
        for preferred in ("zh-CN", "zh-Hans", "zh", "en"):
            if preferred in subs and subs[preferred]:
                lang_code = preferred
                entry = subs[preferred][0]
                break
        else:
            lang_code, items = next(iter(subs.items()))
            if not items:
                return None
            entry = items[0]

        sub_url = entry.get("url")
        if not sub_url:
            return None
        resp = httpx.get(sub_url, timeout=30.0, follow_redirects=True)
        resp.raise_for_status()
        payload = resp.json()
        body = payload.get("body") or []
        segments = [
            TranscriptSegment(
                start=float(s["from"]), end=float(s["to"]), text=s["content"].strip()
            )
            for s in body
        ]
        full_text = " ".join(seg.text for seg in segments)
        return TranscriptResult(
            schema_version=1,
            language=lang_code,
            source="bilibili_cc",
            segments=segments,
            full_text=full_text,
        )

    def fetch_audio(self, meta: VideoMeta, dest: Path) -> None:
        outtmpl = str(dest.with_suffix("")) + ".%(ext)s"
        opts = {
            "format": "bestaudio/best",
            "outtmpl": outtmpl,
            "noplaylist": True,
            "quiet": True,
            "no_warnings": True,
        }
        url = f"https://www.bilibili.com/video/{meta.video_id}"
        self._ytdlp_extract_info(url, opts=opts, download=True)
        if not dest.exists():
            candidates = sorted(dest.parent.glob(f"{dest.stem}.*"))
            candidates = [c for c in candidates if c != dest]
            if not candidates:
                raise RuntimeError(f"yt-dlp produced no audio for {meta.video_id}")
            candidates[0].rename(dest)

    # --- thin seam --------------------------------------------------------

    def _ytdlp_extract_info(
        self, url: str, *, opts: dict[str, Any], download: bool = False
    ) -> dict[str, Any]:
        from yt_dlp import YoutubeDL

        merged = {"quiet": True, "no_warnings": True, **opts}
        with YoutubeDL(merged) as ydl:
            return ydl.extract_info(url, download=download) or {}
```

```python
"""YAML config loading + schema validation + account ID normalization."""
from __future__ import annotations

from pathlib import Path
from typing import Any

import yaml
from pydantic import BaseModel, ConfigDict, Field

from aishelf.scraper.adapters.base import PlatformAdapter
from aishelf.scraper.models import Platform


class AccountConfig(BaseModel):
    model_config = ConfigDict(extra="forbid")

    platform: Platform
    id: str
    display_name: str
    enabled: bool = True
    note: str | None = None


class Defaults(BaseModel):
    model_config = ConfigDict(extra="forbid")

    language_hint: str | None = None
    max_videos_per_account: int = 50
    request_timeout_seconds: int = 30
    min_delay_between_requests_seconds: float = 1.0
    youtube_api_quota_aware: bool = False


class ScraperConfig(BaseModel):
    model_config = ConfigDict(extra="forbid")

    accounts: list[AccountConfig] = Field(min_length=1)
    defaults: Defaults = Field(default_factory=Defaults)


def load_config(path: Path) -> ScraperConfig:
    raw: dict[str, Any] = yaml.safe_load(path.read_text(encoding="utf-8")) or {}
    return ScraperConfig.model_validate(raw)


def resolve_account_ids(
    cfg: ScraperConfig, *, adapters: dict[Platform, PlatformAdapter]
) -> ScraperConfig:
    """Return a copy of cfg with each enabled account's ``id`` normalized to the
    platform's stable identifier. Disabled accounts are left untouched."""
    new_accounts = []
    for acc in cfg.accounts:
        if not acc.enabled:
            new_accounts.append(acc)
            continue
        adapter = adapters[acc.platform]
        resolved = adapter.resolve_account_id(acc.id)
        new_accounts.append(acc.model_copy(update={"id": resolved}))
    return cfg.model_copy(update={"accounts": new_accounts})
```

```python
"""Top-level orchestration for the scraper.

Iterates over enabled accounts, fetches listings in the date range, and
processes each video through the metadata → thumbnail → subtitle/ASR → store
pipeline. Per-video failures are recorded but do not stop the batch.
"""
from __future__ import annotations

import logging
import time
import traceback
from dataclasses import dataclass
from datetime import date, datetime, timezone
from pathlib import Path
from typing import Iterable

from aishelf.scraper.adapters.base import PlatformAdapter
from aishelf.scraper.config import AccountConfig, Defaults, ScraperConfig
from aishelf.scraper.models import (
    FailureRecord,
    FailureStage,
    Platform,
    VideoListing,
)
from aishelf.scraper.store import Store
from aishelf.scraper.transcribe import WhisperApi

logger = logging.getLogger(__name__)


@dataclass(frozen=True)
class RunArgs:
    since: date
    until: date
    platform_filter: Platform | None
    account_filter: list[str] | None  # list of stable account IDs
    force: bool
    dry_run: bool
    retry_failed: bool
    max_videos_per_account: int | None  # override default


@dataclass
class RunSummary:
    accounts_processed: int = 0
    videos_new: int = 0
    videos_skipped: int = 0
    videos_failed: int = 0


def run(
    *,
    cfg: ScraperConfig,
    adapters: dict[Platform, PlatformAdapter],
    transcriber: WhisperApi,
    store: Store,
    args: RunArgs,
) -> RunSummary:
    summary = RunSummary()
    started = time.monotonic()

    for account in cfg.accounts:
        if not account.enabled:
            logger.debug("skipping disabled account=%s/%s", account.platform, account.id)
            continue
        if args.platform_filter and account.platform != args.platform_filter:
            continue
        if args.account_filter and account.id not in args.account_filter:
            continue

        summary.accounts_processed += 1
        adapter = adapters[account.platform]
        limit = args.max_videos_per_account or cfg.defaults.max_videos_per_account

        logger.info(
            "starting account=%s/%s display=%s since=%s until=%s",
            account.platform, account.id, account.display_name, args.since, args.until,
        )

        acc_new = acc_skipped = acc_failed = 0
        try:
            listings = list(adapter.list_videos(
                account_id=account.id, since=args.since, until=args.until, limit=limit,
            ))
        except Exception as e:
            logger.error("list_videos failed for %s/%s: %s", account.platform, account.id, e)
            continue

        logger.info("found %d videos in range", len(listings))

        for listing in listings:
            if args.dry_run:
                logger.info("[dry-run] would process video_id=%s title=%r",
                            listing.video_id, listing.title)
                continue

            result = process_one(
                listing=listing, account=account, adapter=adapter,
                transcriber=transcriber, store=store,
                defaults=cfg.defaults, force=args.force, retry_failed=args.retry_failed,
            )
            if result == "new":
                acc_new += 1
            elif result == "skipped":
                acc_skipped += 1
            elif result == "failed":
                acc_failed += 1

            if cfg.defaults.min_delay_between_requests_seconds > 0:
                time.sleep(cfg.defaults.min_delay_between_requests_seconds)

        logger.info(
            "account summary: found=%d new=%d skipped=%d failed=%d",
            len(listings), acc_new, acc_skipped, acc_failed,
        )
        summary.videos_new += acc_new
        summary.videos_skipped += acc_skipped
        summary.videos_failed += acc_failed

    elapsed = time.monotonic() - started
    logger.info(
        "run summary: accounts=%d new=%d skipped=%d failed=%d elapsed=%.1fs",
        summary.accounts_processed, summary.videos_new,
        summary.videos_skipped, summary.videos_failed, elapsed,
    )
    return summary


def process_one(
    *,
    listing: VideoListing,
    account: AccountConfig,
    adapter: PlatformAdapter,
    transcriber: WhisperApi,
    store: Store,
    defaults: Defaults,
    force: bool,
    retry_failed: bool = False,
) -> str:
    """Process a single video through the full pipeline.

    Returns one of: "new" | "skipped" | "failed".
    """
    platform = account.platform
    account_id = account.id
    video_id = listing.video_id

    # Dedup checks
    if store.is_complete(platform, account_id, video_id) and not force:
        logger.debug("skip already-complete video_id=%s", video_id)
        return "skipped"
    if store.has_failure(platform, account_id, video_id) and not retry_failed:
        logger.debug("skip known-failure video_id=%s", video_id)
        return "skipped"

    stage: FailureStage = "fetch_meta"
    try:
        logger.info("processing video_id=%s title=%r", video_id, listing.title)
        meta = adapter.fetch_meta(
            video_id=video_id,
            account_id=account_id,
            account_display_name=account.display_name,
        )

        stage = "fetch_thumbnail"
        thumb_bytes, thumb_ext = adapter.fetch_thumbnail(meta)

        stage = "fetch_subtitle"
        transcript = adapter.fetch_subtitle(meta)

        if transcript is None:
            stage = "fetch_audio"
            audio_path = store.video_dir(platform, account_id, video_id).with_suffix(".tmp") / "audio.tmp.m4a"
            audio_path.parent.mkdir(parents=True, exist_ok=True)
            adapter.fetch_audio(meta, audio_path)

            stage = "asr"
            transcript = transcriber.transcribe(
                audio_path, language_hint=defaults.language_hint
            )

            audio_path.unlink(missing_ok=True)
            logger.info("transcript source=whisper_api lang=%s", transcript.language)
        else:
            logger.info("transcript source=%s lang=%s", transcript.source, transcript.language)

        stage = "write_files"
        store.write_video(
            meta=meta, transcript=transcript,
            thumb_bytes=thumb_bytes, thumb_extension=thumb_ext,
        )
        logger.info("done video_id=%s", video_id)
        return "new"

    except Exception as e:
        logger.error("failed video_id=%s stage=%s error=%s", video_id, stage, e)
        logger.debug(traceback.format_exc())
        # Clean up tmp dir if any
        tmp = store.video_dir(platform, account_id, video_id).with_suffix(".tmp")
        if tmp.exists():
            import shutil
            shutil.rmtree(tmp, ignore_errors=True)

        fr = FailureRecord(
            schema_version=1, platform=platform, account_id=account_id,
            video_id=video_id,
            url=f"https://www.youtube.com/watch?v={video_id}" if platform == "youtube"
                else f"https://www.bilibili.com/video/{video_id}",
            stage=stage,
            error_class=type(e).__name__,
            error_message=str(e)[:1000],
            retryable=stage in {"fetch_meta", "fetch_thumbnail", "fetch_audio"},
            last_attempt_at=datetime.now(tz=timezone.utc),
            attempt_count=1,  # could read prior count from existing failure; v1 keeps simple
        )
        store.write_failure(fr)
        return "failed"
```

```python
"""Command-line interface for the scraper.

Subcommands:
    run               — fetch new videos for the given date range
    retry-failed      — retry videos in _failures/
    validate-config   — lint authors.yaml
    list-config       — print resolved (normalized) account list
"""
from __future__ import annotations

import argparse
import logging
import sys
from datetime import date, datetime
from pathlib import Path

from pydantic import ValidationError

from aishelf.scraper.adapters.bilibili import BilibiliAdapter
from aishelf.scraper.adapters.youtube import YouTubeAdapter
from aishelf.scraper.config import (
    ScraperConfig,
    load_config,
    resolve_account_ids as _resolve_account_ids,
)
from aishelf.scraper.pipeline import RunArgs, run as _pipeline_run
from aishelf.scraper.store import Store
from aishelf.scraper.transcribe import WhisperApi

logger = logging.getLogger("aishelf.scraper")


def _date(s: str) -> date:
    try:
        return datetime.strptime(s, "%Y-%m-%d").date()
    except ValueError as e:
        raise argparse.ArgumentTypeError(str(e))


def _parse_args(argv: list[str]) -> argparse.Namespace:
    p = argparse.ArgumentParser(prog="python -m aishelf.scraper")
    p.add_argument("--config", type=Path, default=Path("config/authors.yaml"))
    p.add_argument("--data-root", type=Path, default=Path("data"))
    p.add_argument("--verbose", action="store_true")

    sub = p.add_subparsers(dest="command", required=True)

    run_p = sub.add_parser("run", help="fetch new videos")
    run_p.add_argument("--since", type=_date, required=True)
    run_p.add_argument("--until", type=_date, default=None)
    run_p.add_argument("--platform", choices=["youtube", "bilibili", "all"], default="all")
    run_p.add_argument("--account", action="append", default=[],
                       help="filter to specific stable account IDs (repeatable)")
    run_p.add_argument("--force", action="store_true")
    run_p.add_argument("--dry-run", action="store_true")
    run_p.add_argument("--max-videos-per-account", type=int, default=None)

    retry_p = sub.add_parser("retry-failed", help="retry videos in _failures/")
    retry_p.add_argument("--platform", choices=["youtube", "bilibili", "all"], default="all")
    retry_p.add_argument("--account", action="append", default=[])

    sub.add_parser("validate-config", help="lint authors.yaml")
    sub.add_parser("list-config", help="print normalized account list")

    return p.parse_args(argv)


def _build_adapters() -> dict:
    return {"youtube": YouTubeAdapter(), "bilibili": BilibiliAdapter()}


def _configure_logging(verbose: bool) -> None:
    level = logging.DEBUG if verbose else logging.INFO
    logging.basicConfig(
        level=level,
        format="%(asctime)sZ %(levelname)-5s %(message)s",
        datefmt="%Y-%m-%dT%H:%M:%S",
        stream=sys.stdout,
    )


def main(argv: list[str] | None = None) -> int:
    args = _parse_args(argv if argv is not None else sys.argv[1:])
    _configure_logging(args.verbose)

    try:
        cfg = load_config(args.config)
    except (FileNotFoundError, ValidationError) as e:
        sys.stderr.write(f"config error: {e}\n")
        return 1

    if args.command == "validate-config":
        # If load_config succeeded, schema is valid.
        print(f"OK: {len(cfg.accounts)} accounts, defaults={cfg.defaults.model_dump_json()}")
        return 0

    if args.command == "list-config":
        try:
            resolved = _resolve_account_ids(cfg, adapters=_build_adapters())
        except Exception as e:
            sys.stderr.write(f"resolution error: {e}\n")
            return 1
        for acc in resolved.accounts:
            marker = "" if acc.enabled else " [disabled]"
            print(f"{acc.platform:10s} {acc.id:30s} {acc.display_name}{marker}")
        return 0

    # Both `run` and `retry-failed` need resolved config + adapters + pipeline.
    adapters = _build_adapters()
    try:
        resolved = _resolve_account_ids(cfg, adapters=adapters)
    except Exception as e:
        sys.stderr.write(f"resolution error: {e}\n")
        return 1

    store = Store(data_root=args.data_root)
    transcriber = WhisperApi()  # constructs real OpenAI client lazily

    if args.command == "run":
        platform_filter = None if args.platform == "all" else args.platform
        run_args = RunArgs(
            since=args.since,
            until=args.until or date.today(),
            platform_filter=platform_filter,
            account_filter=args.account or None,
            force=args.force,
            dry_run=args.dry_run,
            retry_failed=False,
            max_videos_per_account=args.max_videos_per_account,
        )
        _pipeline_run(
            cfg=resolved, adapters=adapters, transcriber=transcriber,
            store=store, args=run_args,
        )
        # Per spec §5.2: per-video failures are already recorded in _failures/,
        # so we exit 0. Exit code 2 is reserved for account-level failures that
        # v1 does not propagate (catch-and-continue in pipeline.run).
        return 0

    if args.command == "retry-failed":
        platform_filter = None if args.platform == "all" else args.platform
        # Build listings synthetically from failure records, then dispatch run with retry_failed=True
        # For v1, simpler: just iterate failures and call process_one directly.
        from aishelf.scraper.pipeline import process_one
        from aishelf.scraper.config import AccountConfig
        from aishelf.scraper.models import VideoListing
        from datetime import timezone

        index_by_account: dict[tuple[str, str], AccountConfig] = {
            (a.platform, a.id): a for a in resolved.accounts
        }

        retried = 0
        succeeded = 0
        for fr in store.iter_failures():
            if platform_filter and fr.platform != platform_filter:
                continue
            if args.account and fr.account_id not in args.account:
                continue
            account = index_by_account.get((fr.platform, fr.account_id))
            if account is None:
                logger.warning("failure record %s/%s not in config — skipping",
                               fr.platform, fr.account_id)
                continue
            adapter = adapters[fr.platform]
            listing = VideoListing(
                video_id=fr.video_id, title="(retry)",
                published_at=fr.last_attempt_at,
            )
            result = process_one(
                listing=listing, account=account, adapter=adapter,
                transcriber=transcriber, store=store, defaults=cfg.defaults,
                force=False, retry_failed=True,
            )
            retried += 1
            if result == "new":
                succeeded += 1

        print(f"retried={retried} succeeded={succeeded}")
        return 0

    return 1  # unreachable; argparse required=True covers this
```
