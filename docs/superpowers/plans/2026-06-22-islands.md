# 语义孤岛/孤本 (Islands) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a `GET /islands` page that ranks embedded items by their degree in the semantic `edges` graph — 孤本 (degree 0) and 弱连接 (1–2 neighbors) — to surface the most isolated/unique collections.

**Architecture:** A pure builder `islands.build_islands(items, edges)` computes degree/neighbors and splits into lone/weak/connected over plain dicts; a tolerant `islands.load_islands(db_path)` reads embedded items + the `edges` table and feeds the builder (mirrors `timeline.py`). A read-only route renders `islands.html`. Zero LLM, zero new deps, zero DB-schema change, zero new config.

**Tech Stack:** Python 3, FastAPI, Jinja2, sqlite3 (stdlib), pytest.

## Global Constraints

- Reads the **derived DB** (`default_db_path(get_data_dir())`), same as `/graph` / `/timeline` — files stay source of truth, DB is rebuildable.
- **Only analyze embedded items** (`items.embedding IS NOT NULL`); unembedded items are "unknown", not islands, and must be excluded.
- Degree = count of `edges` (undirected, canonical `src < dst`) incident to an id. 孤本 = degree 0; 弱连接 = `1 <= degree <= WEAK_MAX_DEGREE` (constant `= 2`); above that = connected (not listed).
- No new dependency, no new config var, no DB-schema/migration change, no LLM call, no persistence/caching.
- `load_islands` must be **tolerant**: missing/old/broken DB → empty result, never raises (like `timeline.load_timeline`).
- Templates autoescape; item/neighbor links use `/videos/{id}` ⁄ `/blogs/{id}` keyed on `type` (`"video"` → videos, else blogs).
- Two degrade states: `stats.embedded == 0` → "needs `ATLAS_EMBED_*` + sync" notice; embedded but no islands → "彼此连接紧密，没有孤岛 🎉".
- Follow existing module style: `from __future__ import annotations`, docstring matching `timeline.py`'s voice.

    - Returns `{"lone": [{"id","type","title","hook"}], "weak": [{"id","type","title","hook","degree","neighbors":[{"id","type","title"}]}], "stats": {"embedded","lone","weak","connected"}}`.
    - Degree from edges whose both endpoints are in `items` (dirty edges skipped). `lone` = degree 0 sorted by `(title,id)`; `weak` = `1..max` sorted by `(degree,title,id)`, neighbors resolved via the items map and sorted by `(title,id)`.

```python
"""语义孤岛/孤本 (Islands) builder + loader, mirroring timeline.py.

Pure `build_islands` ranks embedded items by their degree in the semantic
`edges` graph — 孤本 (degree 0) and 弱连接 (1..WEAK_MAX_DEGREE neighbors) — to
surface the most isolated collections; tolerant `load_islands` reads the
derived items/edges tables and feeds the builder. No LLM, no persistence —
pure per-request assembly.
"""

from __future__ import annotations

import sqlite3

from aishelf.db import schema

WEAK_MAX_DEGREE = 2


def build_islands(items, edges, *, max_weak_degree: int = WEAK_MAX_DEGREE) -> dict:
    """Split embedded items into 孤本 (degree 0) / 弱连接 (1..max) / connected. Pure."""
    by_id = {it["id"]: it for it in items}
    neighbors: dict = {it["id"]: set() for it in items}
    for src, dst in edges:
        if src != dst and src in by_id and dst in by_id:
            neighbors[src].add(dst)
            neighbors[dst].add(src)

    def _entry(it):
        return {"id": it["id"], "type": it.get("type"),
                "title": it.get("title"), "hook": it.get("hook") or ""}

    lone, weak = [], []
    for it in items:
        deg = len(neighbors[it["id"]])
        if deg == 0:
            lone.append(_entry(it))
        elif deg <= max_weak_degree:
            e = _entry(it)
            nbrs = []
            for nid in neighbors[it["id"]]:
                n = by_id.get(nid)
                if n is not None:
                    nbrs.append({"id": n["id"], "type": n.get("type"), "title": n.get("title")})
            nbrs.sort(key=lambda n: ((n["title"] or ""), n["id"]))
            e["degree"] = deg
            e["neighbors"] = nbrs
            weak.append(e)

    lone.sort(key=lambda e: ((e["title"] or ""), e["id"]))
    weak.sort(key=lambda e: (e["degree"], (e["title"] or ""), e["id"]))

    connected = len(items) - len(lone) - len(weak)
    return {
        "lone": lone,
        "weak": weak,
        "stats": {"embedded": len(items), "lone": len(lone),
                  "weak": len(weak), "connected": connected},
    }


def load_islands(db_path) -> dict:
    """Read embedded items + edges from the derived DB and build the islands view.

    Tolerant of a missing/old/broken DB: returns an empty result, never raises
    (mirrors timeline.load_timeline)."""
    try:
        con = schema.connect(db_path)
        try:
            schema.init_db(con)
            rows = con.execute(
                "SELECT id, type, title, hook FROM items WHERE embedding IS NOT NULL"
            ).fetchall()
            edge_rows = con.execute("SELECT src, dst FROM edges").fetchall()
        finally:
            con.close()
    except sqlite3.Error:
        return build_islands([], [])
    items = [{"id": r["id"], "type": r["type"], "title": r["title"], "hook": r["hook"]}
             for r in rows]
    edges = [(r["src"], r["dst"]) for r in edge_rows]
    return build_islands(items, edges)
```
