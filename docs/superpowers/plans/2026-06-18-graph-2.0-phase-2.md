# 图谱 2.0 Phase 2（B 回放 / D 漫游 / E 路径）Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 给 `/graph` 的极简动词条接上三个"动词"——B 时间回放、D 语义漫游、E 语义路径（含一个新的流式解说后端 `POST /graph/path`），同时保持"静止星系态 + 临时动词"的克制框架。

**Architecture:** B/D/E 都是 `graph.html` 里的前端"模式",共用一个 mode 管理器(同一时刻只一个激活,退出即回到 Phase 1 的静止星系态)。E 额外需要一个流式后端:新纯模块 `aishelf/site/path.py`(`build_messages`)+ `POST /graph/path` 路由,完全镜像 `collide_chat`(`StreamingResponse` + `hermes.sse` + `llm.stream_completion`,用 `db_search.get_by_ids` 水合标题/摘要)。后端纯逻辑 + 路由走 TDD;前端三模式无自动化测试(与现有 `graph.html` 一致,靠手验)。

**Tech Stack:** Python 3 / FastAPI / Pydantic / Jinja2 / Three.js r128(CDN,已在)/ pytest。

**关键事实(已核对当前代码,务必照此设计):**
- **当前 `graph.html` 并不渲染边**:`data.edges` 被 fetch 但从未画成线。所以:
  - **B 回放**只做节点"出生"动画,不涉及边。
  - **D 漫游**用 `data.edges` 计算游走的邻居(数据可用),沿途靠镜头移动叙事。
  - **E 路径**在 `data.edges`(即 `/api/graph` 返回的 top-K 边集)上跑 Dijkstra,然后**自己画一条发光折线**作为高亮轨迹(而非"高亮已有的弧")。这是对 spec 第 3 节 E 描述的一处务实修正——意图不变(轨迹沿语义边),只是因为底图不画边,改由 E 临时画出。
- 节点数据每个都带 `id/type/title/alias/cluster/collected_at/hook/x/y/z`;`collected_at` NOT NULL;`data.clusters` 为 `[{id,name,color,size}]`;边为 `{source,target,weight}`(weight=余弦相似度,越大越近)。
- 世界坐标 = `(n.x*R, n.y*R, n.z*R)`,`R=100`。
- 现有 `#verbPill` 三个按钮目前是 `disabled`;各任务里逐个启用 + 接逻辑。
- `llm.stream_completion(messages)` yield 的就是 `hermes.sse(...)` 形态的 `data: {json}\n\n` 字符串(见 `collide_chat` 的 `yield from`)。SSE 事件含 `{delta}`/`{done}`/`{error}`。
- `db_search.get_by_ids(db_path, ids) -> {id: SearchHit}`;`SearchHit` 有 `id/type/title/author/summary/...`。
- `app.py` 已 `from pydantic import BaseModel, Field`;site 模块以 `from aishelf.site import (...)` 批量导入;`from pathlib import Path`(没有名为 `path` 的变量,可安全新增 `path` 模块导入)。

```python
"""Semantic-path explanation prompt (pure), mirroring collide.py's pure half.

Given the ordered items on a path between two nodes, build the chat messages
that ask the model to explain the conceptual bridge from start to end. The
route is found client-side over the graph edges; this module only phrases the
explanation. Streamed by POST /graph/path via llm.stream_completion.
"""

from __future__ import annotations


def _fmt(i: int, x: dict, n: int) -> str:
    role = "起点" if i == 0 else ("终点" if i == n - 1 else f"第{i}步")
    return f"{role}：{x['title']}（{x.get('type', '')}）— {(x.get('summary') or '')[:80]}"


def build_messages(items: list[dict]) -> list[dict]:
    """System + user messages for explaining a semantic path. `items` are dicts
    with title/summary/author/type, in path order (start → end)."""
    system = (
        "你是一个善于把想法串起来的思考者。用户会给你一条知识路径——从起点内容出发，"
        "经过若干跳，到达终点内容。请解释这条路径如何把起点和终点连起来：每一跳为什么"
        "顺理成章，整条路又揭示了什么共同的暗线。输出恰好两段纯文本：\n"
        "【这条路怎么走通的】顺着路径把相邻两步的关联讲清楚，1-3 句。\n"
        "【它们共享的暗线】这条路背后没明说的那条主线，1-2 句。\n"
        "具体、有锋芒、不要空话套话。只输出这两段纯文本，不要使用 Markdown 语法"
        "（不要 # 或 * 等），不要额外开场白或总结。"
    )
    n = len(items)
    steps = "\n".join(_fmt(i, x, n) for i, x in enumerate(items))
    user = f"路径上的内容依次是：\n{steps}"
    return [{"role": "system", "content": system}, {"role": "user", "content": user}]
```

```python
@app.post("/graph/path")
def graph_path(req: _PathRequest):
    # Ungated: explanation is cheap (like /ask and /collide), not the Hermes path.
    if len(req.ids) < 2:
        raise HTTPException(status_code=400, detail="需要至少两个节点")
    hits = db_search.get_by_ids(default_db_path(get_data_dir()), req.ids)
    ordered = [hits[i] for i in req.ids if i in hits]   # preserve client path order

    def _gen():
        if len(ordered) < 2:
            yield hermes.sse({"error": "这条路径上的内容找不到了"})
            yield hermes.sse({"done": True})
            return
        items_ = [{"title": h.title, "summary": h.summary, "author": h.author, "type": h.type}
                  for h in ordered]
        yield from llm.stream_completion(path.build_messages(items_))

    return StreamingResponse(_gen(), media_type="text/event-stream")
```

```javascript
  MODES.replay = function () {
    const bar = document.getElementById('replayBar');
    const range = document.getElementById('replayRange');
    const playBtn = document.getElementById('replayPlay');
    const dateEl = document.getElementById('replayDate');
    const times = data.nodes.map((n) => Date.parse(n.collected_at) || 0);
    const tmin = Math.min(...times), tmax = Math.max(...times) || (tmin + 1);
    const span = (tmax - tmin) || 1;
    const DURATION = 18000;          // ms for a full sweep
    let p = 0, playing = true, last = performance.now();

    function applyAt(pp) {
      const cutoff = tmin + pp * span;
      for (const m of nodeMeshes) {
        const t = Date.parse(m.userData.collected_at) || 0;
        const born = t <= cutoff;
        const fade = born ? Math.min(1, (cutoff - t) / (span * 0.04 + 1)) : 0;
        m.visible = born;
        m.material.opacity = born ? Math.max(0.2, fade) : 0;
        if (m.userData._label) {
          m.userData._label.visible = born;
          m.userData._label.element.style.opacity = born ? String(Math.max(0.2, fade)) : '0';
        }
      }
      dateEl.textContent = new Date(cutoff).toISOString().slice(0, 10);
      range.value = String(Math.round(pp * 1000));
    }

    bar.classList.add('show');
    frameHook = function () {
      if (playing) {
        const now = performance.now();
        p += (now - last) / DURATION; last = now;
        if (p >= 1) { p = 1; playing = false; playBtn.textContent = '▶'; }
        applyAt(p);
      } else { last = performance.now(); }
    };
    playBtn.onclick = function () {
      if (p >= 1) p = 0;             // restart from the beginning
      playing = !playing; playBtn.textContent = playing ? '⏸' : '▶';
      last = performance.now();
    };
    range.oninput = function () {
      playing = false; playBtn.textContent = '▶';
      p = Number(range.value) / 1000; applyAt(p);
    };
    applyAt(0);
    return function exit() {
      bar.classList.remove('show');
      playBtn.onclick = null; range.oninput = null;
    };
  };
```

```javascript
      if (cn && n.cluster !== lastCluster) { html += '进入「<b>' + cn + '</b>」　'; lastCluster = n.cluster; }
      html += '<b>' + (n.alias || n.title) + '</b>';
      if (n.hook) html += '<br>' + n.hook;
      cap.innerHTML = html;
      const keep = new Set([id, ...((adj[id] || []).map((e) => e.id))]);
      for (const m of nodeMeshes) {
        const on = keep.has(m.userData.id);
        m.material.opacity = on ? 1 : 0.12;
        if (m.userData._label) m.userData._label.element.style.opacity = on ? '1' : '0.12';
      }
    }
    function next() {
      const nbrs = (adj[cur] || []).map((e) => e.id);
      const fresh = nbrs.filter((i) => !visited.has(i));
      const pool = fresh.length ? fresh : nbrs;
      if (pool.length) return pool[Math.floor(Math.random() * pool.length)];
      const rest = ids.filter((i) => !visited.has(i));   // dead end -> jump
      return (rest.length ? rest : ids)[Math.floor(Math.random() * (rest.length ? rest.length : ids.length))];
    }

    go(cur);
    frameHook = function () {
      const now = performance.now(); const dt = now - last; last = now;
      const p = posOf[cur];
      if (p) {
        controls.target.lerp(p, 0.05);
        target.copy(p).normalize().multiplyScalar(R * 2.3);
        camera.position.lerp(target, 0.03);
      }
      hold -= dt;
      if (hold <= 0) { hold = HOLD; go(next()); }
    };
    return function exit() { cap.classList.remove('show'); };
  };
```

```javascript
  MODES.path = function () {
    const panel = document.getElementById('pathPanel');
    const hint = document.getElementById('pathHint');
    const text = document.getElementById('pathText');
    panel.classList.add('show'); hint.textContent = '点第一颗星'; text.textContent = '';
    let a = null, lineObj = null;

    function dijkstra(src, dst) {
      const dist = { [src]: 0 }, prev = {}, seen = new Set();
      const pq = [[0, src]];
      while (pq.length) {
        pq.sort((x, y) => x[0] - y[0]);
        const [d, u] = pq.shift();
        if (u === dst) break;
        if (seen.has(u)) continue; seen.add(u);
        for (const e of (adj[u] || [])) {
          const nd = d + (1 - e.w);          // stronger edge (higher w) = shorter
          if (dist[e.id] == null || nd < dist[e.id]) { dist[e.id] = nd; prev[e.id] = u; pq.push([nd, e.id]); }
        }
      }
      if (dst !== src && prev[dst] == null) return null;
      const route = [dst]; let c = dst;
      while (c !== src) { c = prev[c]; if (c == null) return null; route.unshift(c); }
      return route;
    }
    function clearLine() {
      if (lineObj) { scene.remove(lineObj); lineObj.geometry.dispose(); lineObj.material.dispose(); lineObj = null; }
    }
    function drawRoute(route) {
      clearLine();
      const pts = route.map((id) => posOf[id]).filter(Boolean);
      lineObj = new THREE.Line(new THREE.BufferGeometry().setFromPoints(pts),
        new THREE.LineBasicMaterial({ color: 0x6cf0ff, transparent: true, opacity: 0.9 }));
      scene.add(lineObj);
      const keep = new Set(route);
      for (const m of nodeMeshes) {
        const on = keep.has(m.userData.id);
        m.material.opacity = on ? 1 : 0.08;
        if (m.userData._label) m.userData._label.element.style.opacity = on ? '1' : '0.08';
      }
    }
    async function explain(route) {
      try {
        const resp = await fetch('/graph/path', {
          method: 'POST', headers: { 'Content-Type': 'application/json' },
          body: JSON.stringify({ ids: route }),
        });
        const reader = resp.body.getReader(), dec = new TextDecoder();
        let buf = ''; text.textContent = '';
        while (true) {
          const { done, value } = await reader.read(); if (done) break;
          buf += dec.decode(value, { stream: true });
          const lines = buf.split('\n'); buf = lines.pop();
          for (const ln of lines) {
            if (!ln.startsWith('data: ')) continue;
            const ev = JSON.parse(ln.slice(6));
            if (ev.delta) text.textContent += ev.delta;
            if (ev.error) text.textContent = ev.error;
          }
        }
      } catch (e) { text.textContent = '解说服务暂时不可用'; }
    }
    function onPick(ev) {
      const rect = renderer.domElement.getBoundingClientRect();
      mouse.x = ((ev.clientX - rect.left) / rect.width) * 2 - 1;
      mouse.y = -((ev.clientY - rect.top) / rect.height) * 2 + 1;
      ray.setFromCamera(mouse, camera);
      const hit = ray.intersectObjects(nodeMeshes)[0];
      if (!hit) return;
      const id = hit.object.userData.id;
      if (a == null) { a = id; hint.textContent = '已选起点，再点一颗终点星'; return; }
      if (id === a) return;
      const route = dijkstra(a, id);
      if (!route) { hint.textContent = '这两颗星之间暂无语义通路，换一对试试'; a = null; return; }
      hint.textContent = '路径：' + route.length + ' 颗星 · 解说中…';
      drawRoute(route); explain(route); a = null;
    }
    renderer.domElement.addEventListener('click', onPick);
    return function exit() {
      panel.classList.remove('show'); clearLine();
      renderer.domElement.removeEventListener('click', onPick);
    };
  };
```
